# OurChat 状态码

**_注：任何状态中都可能包含INTERNAL,请务必做好校验_**
**_由于服务器随时可能进入维护状态，FAILED_PRECONDITION 随时可能出现_**

**_以上特殊信息不再赘述_**

## 状态码

请参考 [grpc 文档](https://grpc.io/docs/guides/status-codes/)

## 各种信息下的状态码具体解释

### 获取会话信息返回信息

| code | meaning      |
| :--- | :----------- |
| 0    | 获取成功     |
| 3    | 该会话不存在 |

### 注册返回信息

| code | meaning    |
| :--- | :--------- |
| 0    | 注册成功   |
| 4    | 邮箱已存在 |
| 10   | 未完成验证 |

### 登录返回信息

| code | meaning       |
| :--- | :------------ |
| 0    | 登录成功      |
| 10   | 账号/密码错误 |

### 新建会话返回信息

| code | meaning              |
| :--- | :------------------- |
| 0    | 新建成功             |
| 6    | 到达创建会话数量上限 |

### 获取账号信息返回信息

| code | meaning      |
| :--- | :----------- |
| 0    | 获取成功     |
| 3    | 该账号不存在 |

### 获取服务器状态

| code | meaning      |
| :--- | :----------- |
| 0    | 服务器正常   |
| 2    | 服务器维护中 |

### 验证状态

| code | meaning    |
| :--- | :--------- |
| 0    | 验证通过   |
| 7    | 验证超时   |

### 注销返回信息

| code | meaning    |
| :--- | :--------- |
| 0    | 注销成功   |
| 10   | 未完成验证 |

### 设置账号信息返回信息

| code | meaning    |
| :--- | :--------- |
| 0    | 设置成功   |

### 上传文件返回信息

| code | meaning                      |
| :--- | :--------------------------- |
| 0    | 上传成功                     |
| 6    | 因账号相关限制导致请求被拒绝 |

### 邮件发送

| code | meaning                  |
| :--- | :----------------------- |
| 0    | 上传成功                 |
| 3    | 请求邮箱无法发送成功     |
| 10   | 请求被拒绝               |
| 11   | 对应功能不存在或无法启用 |

### 同意加入会话返回信息

| code | meaning        |
| :--- | :------------- |
| 0    | 同意成功       |
| 10   | 请求被拒绝     |
