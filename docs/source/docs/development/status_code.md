# OurChat 状态码

## 状态码表格

| code | meaning        |
| :--- | :------------- |
| 0    | 运行成功       |
| 1    | 服务器错误     |
| 2    | 服务器维护中   |
| 3    | 请求信息不存在 |
| 4    | 新增信息已存在 |
| 5    | 参数错误       |
| 6    | 账号限制       |
| 7    | 等待超时       |
| 8    | 未知错误       |
| 9    | 请求被拒绝     |
