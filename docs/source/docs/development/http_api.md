# Http Api

## 文件上传

接口:`/upload`
请求类型:`POST`

| 请求头 | 说明     |
| :----- | :------- |
| Key    | 资源密钥 |

| 状态码              | 说明                 |
| :------------------ | :------------------- |
| NotFound            | 请求上传的资源不存在 |
| InternalServerError | 服务器内部错误       |
| BadRequest          | 哈希不匹配等         |
| Ok                  | 成功                 |

直接上传文件内容即可

**关于获取该接口，参见[交换格式](./msg_json.md#文件上传请求)**

## 文件下载

接口:`/download`
请求类型:`POST`

| 请求头 | 说明     |
| :----- | :------- |
| Key    | 资源密钥 |

| 状态码              | 说明                 |
| :------------------ | :------------------- |
| NotFound            | 请求下载的资源不存在 |
| InternalServerError | 服务器内部错误       |
| BadRequest          | 资源密钥不存在       |
| Ok                  | 成功                 |

body 中包含了文件的数据
